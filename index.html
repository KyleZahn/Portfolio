Whisker

A Tinder-themed app that utilizes the Petfinder API. Users are able to search for pets up for adoption in any zip code while filtering their search by an animals age, breed, type (cat, dog, etc), size, and more. When a user 'swipes right' on a pet, the pet is added to their favorites in AsyncStorage to view all of the pet's photos, their description, shelter contact information, and location. Created with React Native. View the source code here.
